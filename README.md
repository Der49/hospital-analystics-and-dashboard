Deploying to Streamlit Community Cloud

This is the "set my website up" part — connecting your existing GitHub repo to Streamlit's free hosting.

1. Go to share.streamlit.io
2. Click Sign up / Continue with GitHub → authorize it to access your GitHub account
3. Click Create app (or New app)
4. Fill in:
    Repository: select your repo from the dropdown (e.g., hospital-analytics-dashboard)
    Branch: main
    Main file path: app.py
    App URL (optional, customizable): something like hospital-analytics-yourname → gives you https://hospital-analytics-yourname.streamlit.app
5. Click Deploy
